Suhani Sketch Studio
A beautiful handcrafted portrait sketch studio website — pure HTML, CSS & JavaScript. No frameworks, no dependencies.
Files
File	Description
`index.html`	Complete website (all HTML, CSS & JS in one file)
`couple-sketch.jpg`	Couple sketch gallery image
`solo-sketch.jpg`	Single portrait gallery image
`family-sketch.jpg`	Family sketch gallery image
`dog-sketch.jpg`	Pet sketch gallery image
Features
Beautiful hero section with real sketch images
Gallery with 4 categories: Single, Couple, Family, Pet
Celebrity tab with special modal message
Live Photo-to-Sketch converter (Canvas API)
5-step order form that emails to both addresses
Pricing section, testimonials, Instagram section
Fully responsive (mobile + desktop)
Hosting on GitHub Pages
Create a new GitHub repository
Upload all 5 files (index.html + 4 images)
Go to Settings → Pages → Source: Deploy from branch → `main` / `root`
Your site will be live at `https://yourusername.github.io/repo-name`
